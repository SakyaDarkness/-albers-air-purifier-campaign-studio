# 艾伯斯空气净化器宣传生成器

一个无依赖的本地 Web 工具，用于生成空气净化器项目的宣传图文草稿、海报文案、竞品对比结论、互动评论话题、官方评论回复和二级回复。

## 使用方式

推荐启动本地服务后访问：

```bash
node dev-server.mjs
```

然后打开：

```text
http://127.0.0.1:5173/index.html
```

也可以直接用浏览器打开 `index.html`，但部分浏览器可能限制复制、下载或导入等本地文件能力。

## 功能

- 录入项目名称和行动引导语
- 录入承接方式和主推优惠，用于私信/客服跟进话术
- 录入预算、点击成本、线索率、成交率、客单价和毛利率
- 管理本地项目库，可保存、载入、复制、删除多个活动项目
- 管理 CRM 线索，记录来源、关注点、状态和下一步动作
- 调整海报配色和版式
- 生成项目总览，汇总完备度、成本、投放、竞品和下一步动作
- 生成销售简报，方便导购、客服或朋友快速掌握产品话术
- 录入品牌、型号、目标人群、场景、CADR、适用面积、噪音、价格和核心卖点
- 录入滤芯价格、滤芯寿命、功率、每日使用时长、电价和质保
- 维护合规风险词库，用于发布前扫描绝对化或高风险表达
- 可添加、编辑、删除竞品参数，并记录参数来源和核对日期
- 管理证据素材清单，记录检测报告、竞品截图、价格截图等文件信息
- 一键生成小红书、朋友圈、抖音图文风格的营销文案
- 生成多版本标题、开头、CTA 和封面字
- 生成海报预览和可下载 SVG 海报
- 生成 AI 生图提示词，方便交给生图工具做宣传主视觉
- 生成评论区互动话题、官方回复、二级回复
- 生成评论矩阵，覆盖种草、提问、理性对比和官方承接
- 竞品对比结论只基于当前录入参数生成
- 生成可视化竞品参数表
- 生成年使用成本对比，包含年滤芯成本、年电费和总年成本
- 生成 7 天发布排期，包含主题、内容、素材和审核点
- 生成拍摄清单与 30 秒短视频分镜
- 生成线索承接话术，覆盖参数表、户型面积、竞品对比、滤芯成本和优惠咨询
- 生成投放 KPI 预估，包含点击、线索、成交、销售额、毛利和 ROAS
- 生成投放复盘模板，方便记录实际数据、内容表现和下轮动作
- 生成执行看板，按运营、内容、拍摄、客服、投放、复盘拆分任务
- 生成一页汇报，适合开会、发群或打印
- 一键导出竞品、成本、KPI、任务和素材清单 CSV
- 导出 Excel 兼容 `.xls` 表格
- 生成数据仪表盘和平台发布检查
- 保存当前项目到浏览器本地存储
- 一键导出全部文案为 TXT
- 导出和导入项目 JSON，方便迁移或复用活动配置
- 生成发布审核清单，提示缺失参数、来源日期和常见绝对化用词风险

## 发布注意

竞品参数、检测报告、售价、滤芯等级、售后政策都可能随时间变化。正式发布前需要核对来源、日期和具体型号，避免夸大或不实表述。

证据素材功能只保存文件名、类型、大小、备注和添加日期，不会把文件内容写入项目 JSON。正式归档时请把原始截图、报告文件和导出的项目 JSON 放在同一资料夹中。
